# Spendly - Personal Finance Management System

A Python-based personal finance management application developed as part of the **Software Architecture (4SA3)** course.

The project was designed by applying software architecture principles, design patterns, and clean code practices to create a maintainable and scalable application for tracking personal expenses.

---

## Project Overview

Spendly helps users manage their finances by recording expenses, organizing transactions, and generating financial summaries.

The project focuses on software architecture rather than simply building an application. It demonstrates the implementation of layered architecture, separation of concerns, and design patterns.

---

## Features

- Record and manage expenses
- Category-based expense tracking
- Monthly financial summaries
- Firebase database integration
- Currency conversion support
- Strategy Pattern for report generation
- MVC (Model-View-Controller) architecture

---

## Technologies Used

- Python
- Firebase Firestore
- REST API Integration
- MVC Architecture
- Strategy Design Pattern
- Object-Oriented Programming (OOP)
- Draw.io (Architecture Modeling)
